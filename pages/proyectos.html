Presentación de proyectos

En esta sección comparto algunos proyectos y prácticas realizados durante mi aprendizaje en desarrollo web, aplicando conceptos de HTML5, estructura semántica y organización de contenido.

Proyectos

Sitio web personal

Este proyecto consiste en la creación de un sitio web personal utilizando HTML5 y etiquetas semánticas. El objetivo fue practicar la estructura básica de una página web y la navegación entre diferentes archivos HTML.

Página de práctica HTML

Esta práctica estuvo enfocada en el uso de etiquetas HTML, listas, imágenes y organización correcta del contenido para mejorar la estructura y accesibilidad de las páginas web.

Diseño responsive

En este proyecto trabajé conceptos relacionados con diseño responsive y adaptación de contenido para distintos dispositivos y tamaños de pantalla.

Aprendizajes obtenidos

Gracias a estos proyectos pude mejorar mis conocimientos en desarrollo web, estructura semántica, uso de Git y GitHub, y organización de contenido en sitios web modernos.
